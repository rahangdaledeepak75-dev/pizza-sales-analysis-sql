# 🍕 Pizza Sales Analysis (SQL Project)

## 📌 Project Overview
This project focuses on analyzing pizza sales data using SQL to extract meaningful business insights. The goal is to understand sales patterns, customer behavior, and product performance.

## 🎯 Objectives
- Identify top-selling pizzas
- Analyze order patterns by time
- Evaluate sales performance
- Generate insights for business decision-making

## 🛠️ Tools & Technologies
- SQL (PostgreSQL)
- Advanced Excel
- Canva

## 📊 Analysis Performed
- Total orders and revenue calculation
- Top and least selling pizzas
- Hour-wise order distribution
- Category-wise sales analysis
- Average order value analysis

## 📁 Files Included
- `Pizza_Sales_Analysis_Presentation.pdf` → Complete project presentation
- `Queries.sql` → All SQL queries used for analysis

## 🚀 Key Insights
- Peak ordering hours identified during high-demand periods
- Certain pizza types consistently perform better than others
- Sales distribution varies across different categories

This project demonstrates my ability to work with structured data using SQL, perform detailed analysis, and derive actionable insights. It reflects my strong foundation in data analysis and problem-solving.
